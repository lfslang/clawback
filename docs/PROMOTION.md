### 完整版（适合知乎/掘金/V2EX）

## 我开源了一个 OpenClaw 配置管理工具 - Clawback

### 为什么开发这个工具？

在使用 OpenClaw 的过程中，我发现配置管理是个痛点：
- 配置改错了无法快速恢复
- 没有版本历史
- 多环境配置混乱

于是开发了 Clawback，一个专注于 OpenClaw 配置备份与回滚的 TUI 工具。

### 核心特性

🌍 **多语言支持**
- 支持 7 种语言：中/英/日/德/西/法/韩
- 界面完全本地化

🔄 **一键回滚**
- 创建带标签的快照
- 对比功能预览变更
- 秒级回滚到任意版本

🧠 **智能备份**
- 自动检测配置变化
- 避免重复备份
- 支持定时自动备份

🔒 **安全保护**
- API Key 等敏感信息自动打码
- 快照文件权限保护

### 安装使用

```bash
# 一键安装
curl -fsSL https://raw.githubusercontent.com/lfslang/clawback/main/install-remote.sh | bash

# 启动
clawback
```


- 纯 Bash 实现，无依赖
- 模块化架构，易于扩展
- 完整的错误处理和日志
- 符合 Shell 脚本最佳实践

### Full Version (Reddit/HN)

## Show HN: Clawback - Multi-language TUI for OpenClaw Config Backup

### The Problem

Managing OpenClaw configurations can be painful:
- No easy way to rollback bad changes
- No version history
- Multiple environment configs get messy

### The Solution

Clawback - A focused TUI tool for OpenClaw configuration backup and rollback.

### Key Features

🌍 **Multi-language Support**
- 7 languages: EN/ZH/JA/DE/ES/FR/KO
- Fully localized interface

🔄 **One-click Rollback**
- Create labeled snapshots
- Preview changes with diff
- Instant rollback to any version

🧠 **Smart Backup**
- Auto-detect config changes
- Avoid duplicate backups
- Scheduled auto-backup support

🔒 **Security**
- Auto-mask sensitive data (API keys)
- Protected snapshot file permissions

### Installation

```bash
# One-line installer
curl -fsSL https://raw.githubusercontent.com/lfslang/clawback/main/install-remote.sh | bash

# Run
```
