# 快速参考

## 常用命令

```bash
# 启动
clawback

# 帮助
clawback --help

# 详情
clawback --details

# 卸载
~/.local/share/clawback/uninstall.sh
```

## 快捷键

| 键 | 功能 |
|----|------|
| ↑/↓ | 菜单导航 |
| Enter | 确认选择 |
| ESC | 返回/退出 |
| Q | 返回 |
| V | 查看模式 |
| R | 回滚 |
| D | 删除 |
| C | 对比 |

## 语言代码

| 代码 | 语言 |

| es | Español |

## 环境变量

```bash
export CLAWBACK_LANG=zh    # 设置语言
```

## 目录

| 路径 | 说明 |
|------|------|
| ~/.local/share/clawback/ | 程序目录 |
| ~/.openclaw/.snapshots/ | 快照目录 |
| ~/.local/bin/clawback | 命令链接 |
