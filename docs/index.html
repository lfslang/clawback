Clawback

OpenClaw 配置回滚工具 - 多语言 TUI（终端用户界面）

Bash TUI Multi-language MIT

特性

🌍多语言支持 - 7种语言：中/英/日/德/西/法/韩
🔄一键回滚 - 秒级回滚到任意版本
🧠智能备份 - 自动检测配置变化
🔒安全保护 - 敏感信息自动打码
